Inception

ONE CONTAINER IS NOT ENOUGH, WE NEED MORE!

Description

The Inception project is a hands-on introduction to building and deploying a full infrastructure setup using containerization. The objective is to create a fully functional environment, complete with services like web servers, databases, and network configurations, all orchestrated within Docker containers.
